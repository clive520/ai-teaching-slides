什麼是 AI Agent？

有手有腳的自動化助理 ── 國中小教師實務應用

📋 行政溝通 ｜ 👨‍🏫 個人化家教 ｜ 📈 數據預警

👉 教學指引 (封面)

各位老師，我們已經了解生成式 AI (超級大腦) 的威力，但如果這個大腦長出了手腳，能自己幫你做事呢？這就是今天的主角：AI Agent 代理智能體！

AI Agent 的核心概念

不僅是「問答」，而是能「規劃步驟、呼叫工具、完成任務」。

生成式 AI (ChatGPT)

  • 一問一答，依賴指令推進
  • 只會給步驟，不會幫你做
  • 例如：給你出國旅遊清單

AI Agent 代理體

  • 給大目標，它自行拆解步驟
  • 會自己查資料、甚至幫你訂票
  • 具備長期記憶與工具使用能力

領域一：行政與親師溝通自動化

工具：LINE Bot + Dify/Coze

教學痛點

  • 每天下班還要回覆家長同樣的問題 (例如明天的午餐費)。
  • 新住民家長有語言隔閡，溝通耗時。

班級專屬客服 Agent

  • 流程：家長傳 LINE，Agent 自動查詢雲端出缺席表並回覆。
  • 功能：內建自動翻譯，泰文/印尼文也能無縫溝通。
  • 👉 讓老師奪回下班後的休息時間！

領域二：個人化專屬家教 (Socratic Tutor)

工具：ChatGPT Custom GPTs / NotebookLM

教學痛點

  • 一班 30 人，無法針對落後學生進行一對一指導。
  • 學生用普通 AI，通常只會「直接抄答案」。

蘇格拉底引導 Agent

  • 設定：在系統後台強制設定「絕對不給直接解答」。
  • 互動：當學生問數學，Agent 會給出「提示與反問」，引導學生自己思考。
  • 👉 真正達成適性揚才的因材施教。

領域三：早期預警與數據追蹤

工具：Make / Zapier 搭配 API

教學痛點

  • 往往等到段考成績出來，才知道學生已經放棄學習。
  • 無法每天肉眼檢查每個學生的線上學習平台登入紀錄。

學習狀況預警系統

  • 自動監控：Agent 自動爬取線上平台紀錄，發現「連續未交作業」或「作答時間異常」。
  • 主動通知：主動發送警示 Email 給老師，建議盡早介入關心。
  • 👉 用科技補足教師的眼睛，降低中輟率。

Agent 導入的隱私與安全指引

資料隱私 (FERPA 規範)

保護學生個資

  • 絕對不可將學生真實姓名與成績丟給公開的 AI。
  • 請使用去識別化的代號，或採用校園版企業帳號。

Human-in-the-Loop

人類必須在迴圈內

  • Agent 只能做「初篩與草稿」。
  • 發送成績單或家長信前，絕對需要老師進行最後審核。

先小規模測試

循序漸進導入

  • 建議先挑選單一繁雜業務 (例如家長常見 Q&A) 讓 Agent 試水溫。
  • 確保準確率 99% 後再推廣至全校。
