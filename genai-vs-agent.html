生成式 AI vs AI Agent

從「對話框」到「真實世界」的進化

👉 教學指引 (開場)

1. 破冰引言：各位老師/同學，我們每天都在用 ChatGPT，但你知道我們用的只是 AI 的「冰山一角」嗎？

2. 課程目標：今天我們不只要認識「生成式 AI」，更要帶大家看懂未來趨勢——「AI Agent (AI 代理人)」。

生成式 AI：內容的創造者

  • 定位：數位大腦、百科全書、專業創作者。
  • 工作模式：反應型 (Reactive)。你給提示詞 (Prompt)，它給你回應。
  • 能力極限：任務在「產出文字、圖片或程式碼」後即結束。
  • 它沒有手腳，無法幫你「執行」真實世界的動作。

👉 教學指引 (生成式 AI)

1. 比喻法：生成式 AI 就像是一個坐在書桌前、絕頂聰明的軍師。你問他什麼他都知道，但他不會自己站起來去幫你倒水。

2. 生活舉例：請 ChatGPT 寫一封英文信。它寫得很好，但你還是得自己「複製、打開 Gmail、貼上、按發送」。

AI Agent：目標的執行者

  • 定位：全自動數位助理。
  • 工作模式：主動型 (Proactive)。給定「大目標」，它會自主規劃。
  • 核心能力：
    • 感知環境：讀取外部資訊
    • 使用工具 (Tool Use)：呼叫 API、上網搜尋
    • 自主行動：規劃步驟並執行

👉 教學指引 (AI Agent)

1. 比喻法延續：AI Agent 不只有大腦，還長出了「手跟腳」。它不僅能思考，還能操作電腦。

2. 生活舉例：同樣是英文信。你只要說「幫我回信給客戶說我明天休假」，Agent 會自動讀取最新郵件、擬定草稿，然後「自己透過 Gmail 幫你寄出去」。

核心差異對比

比較項目 生成式 AI (GenAI) AI Agent (AI 代理人)
核心目標 創作內容 (文字/圖片) 完成任務 (執行工作流)
工作模式 一問一答 (反應型) 自主規劃 (主動型)
自主性 低 (依賴人類明確指令) 高 (能自主判斷並調整策略)

👉 教學指引 (表格比較)

1. 引導觀看表格：帶著學生掃過三個維度 (目標、模式、自主性)。

2. 重點強調：自主性是最大的分水嶺。未來的 AI 發展方向，就是從左邊走向右邊。

兩者的完美協作

它們並非互斥，而是最強大的組合！

生成式 AI = 大腦 (邏輯推理)
+
AI Agent = 手腳 (工具執行)

需要「靈感創意」找生成式 AI；需要「完成繁瑣工作」找 AI Agent。

👉 教學指引 (總結)

1. 觀念釐清：AI Agent 的核心運算能力，其實還是來自於背後的生成式 AI (如 GPT-4)。沒有聰明的大腦，手腳就會做錯事。

2. 收尾提問：「大家能想到有什麼日常工作，是你想交給 Agent 幫你自動執行的嗎？」(與台下互動)
